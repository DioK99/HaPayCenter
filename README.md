### 项目介绍
***

- `HaPayCenter基于Spring Cloud架构开发的聚合支付系统（四方支付系统），已对接支付宝的APP支付，WAP支付，PC支付，当面付等。具有代理体系，同一商户下同渠道的轮询机制，支付日限量、次限量等。可直接用于生产环境。

### 项目结构
***
```
HaCenter
├── HaApiCenter -- 聚合支付中心
|    ├── HaAdmin -- 运营管理后台
|    ├── HaApi -- 接口模块
|    ├── HaConfig -- 配置中心
|    ├── HaGateway -- 网关
|    ├── HaPayService -- 支付服务中心
|    ├── HaServer -- 服务注册中心
├── HaModel -- 数据持久层
├── HaPub -- 公共模块
├── HaSDK -- 接口测试工程
```

项目启动顺序：
```
HaServer > HaConfig > HaService > HaApi > HaGateway
```

***
